# Лабораторная работа №1: Распознавание образов на основе контролируемого обучения

## Цель работы
[cite_start]Изучить особенности методов распознавания образов, использующих контролируемое обучение, и научиться классифицировать объекты с помощью алгоритма К-средних[cite: 3].

## Теоретические основы
[cite_start]В основе алгоритма лежит понятие «расстояние», выступающее критерием в ходе принятия решений[cite: 10]. 
Главная особенность реализации:
* [cite_start]Наличие априорных сведений о количестве классов (K)[cite: 7].
* [cite_start]Использование правила минимального расстояния для отнесения вектора к классу[cite: 20, 21].
* [cite_start]Оптимизация ядер классов через минимизацию среднеквадратичного отклонения (поиск среднего арифметического)[cite: 24, 25, 26].



## Описание реализации
Программа написана на языке Python и позволяет:
1. [cite_start]Задавать количество объектов (от 1 000 до 100 000) и классов (от 2 до 20) через консоль[cite: 13].
2. [cite_start]Генерировать случайные координаты объектов[cite: 13].
3. [cite_start]Визуализировать процесс: от начального распределения до финальной стабилизации ядер[cite: 16, 18].
