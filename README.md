# Лабораторная работа №1: Распознавание образов на основе контролируемого обучения(Completed with AI)

## Цель работы
Изучить особенности методов распознавания образов, использующих контролируемое обучение, и научиться классифицировать объекты с помощью алгоритма К-средних.

## Теоретические основы
В основе алгоритма лежит понятие «расстояние», выступающее критерием в ходе принятия решений. 
Главная особенность реализации:
* Наличие априорных сведений о количестве классов (K).
* Использование правила минимального расстояния для отнесения вектора к классу.
* Оптимизация ядер классов через минимизацию среднеквадратичного отклонения (поиск среднего арифметического).



## Описание реализации
Программа написана на языке Python и позволяет:
1. Задавать количество объектов (от 1 000 до 100 000) и классов (от 2 до 20) через консоль.
2. Генерировать случайные координаты объектов.
3. Визуализировать процесс: от начального распределения до финальной стабилизации ядер.
